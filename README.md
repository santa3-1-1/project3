鸢尾花数据分类与可视化项目

项目简介

本项目是基于Python的机器学习可视化分析项目，使用鸢尾花数据集进行数据分类和多种可视化展示。项目包含2D/3D数据可视化、决策边界分析、概率分布展示以及多模型性能评估。

项目特色

🎯 核心功能

• 2D数据可视化：散点图、决策边界、概率分布

• 3D交互可视化：支持360°旋转查看的3D决策边界和概率分布

• 多模型对比：5种分类器的综合性能评估

• 深度分析：特征重要性、ROC曲线、混淆矩阵等

🔬 技术栈

• Python 3.x

• scikit-learn

• Matplotlib / Seaborn

• Plotly（交互式可视化）

• NumPy / Pandas
